# 프로젝트 로드맵

---

## Milestone 1: PoC (1~12주)

- **목표**: GitHub 연동 + Python SAST + 자동 패치 PR의 엔드투엔드 플로우 동작 증명
- **기능**: F-01, F-02, F-03, F-04
- **완료 기준**:
  - GitHub App 설치 후 저장소 연동 및 PR 이벤트 기반 자동 스캔 동작
  - Python 코드 대상 SQL Injection, XSS, Hardcoded Credentials 탐지 가능
  - 탐지된 취약점에 대한 패치 PR 자동 생성 동작
  - 탐지 정확도 80% 이상 (오픈소스 벤치마크 기준)
  - 패치 PR 승인율 40% 이상
  - 파일럿 고객 3사 온보딩 및 피드백 수집 완료

### 상세 일정

| 주차 | 작업 내용 | 관련 기능 |
|------|-----------|-----------|
| Week 1-2 | Semgrep 룰셋 구성 + Python AST 분석 파이프라인 구축 | F-02 |
| Week 3-4 | Claude API 연동: 오탐 필터 + 패치 코드 생성 프로토타입 | F-02, F-03 |
| Week 5-6 | GitHub App 개발 + Webhook 수신 + 자동 PR 생성 연동 | F-01, F-03 |
| Week 7-8 | 스캔 결과 API + 기본 웹 UI + 내부 테스트 (벤치마크) | F-04 |
| Week 9-10 | 파일럿 고객 3사 온보딩 + 실사용 테스트 | 전체 |
| Week 11-12 | 피드백 반영 + 버그 수정 + 데모 영상 제작 | 전체 |

## Milestone 2: Beta (13~26주)

- **목표**: 다국어 지원 + 오탐 관리 + 대시보드로 제품 완성도 확보, 유료 전환 시작
- **기능**: F-05, F-06, F-07, F-08
- **완료 기준**:
  - JavaScript/TypeScript, Java, Go 언어 탐지 지원
  - OWASP Top 10 전체 카테고리 탐지 가능
  - 오탐 마킹 -> 학습 -> 자동 제외 파이프라인 동작
  - 저장소별/팀별 보안 점수 대시보드 제공
  - Slack/Teams 알림 연동 동작
  - 탐지 정확도 85% 이상
  - 유료 고객 10사 전환

### 상세 일정

| 주차 | 작업 내용 | 관련 기능 |
|------|-----------|-----------|
| Week 13-16 | JS/TS, Java, Go 언어 Semgrep 룰셋 확장 + LLM 프롬프트 조정 | F-05 |
| Week 15-18 | 오탐 피드백 시스템 설계 + 구현 (마킹, 학습, 자동 제외) | F-06 |
| Week 17-20 | 대시보드 프론트엔드 개발 (보안 점수, 추이 그래프, 필터링) | F-07 |
| Week 19-22 | Slack/Teams Webhook 연동 + 알림 설정 UI | F-08 |
| Week 23-26 | 통합 테스트 + 베타 고객 확대 + 안정화 | 전체 |

## Milestone 3: v1.0 (27~52주)

- **목표**: 멀티 플랫폼 + CISO 리포트 + IDE 플러그인으로 엔터프라이즈 대응 완성
- **기능**: F-09, F-10, F-11
- **완료 기준**:
  - GitHub/GitLab/Bitbucket 3개 플랫폼 연동 지원
  - CISO용 PDF 리포트 + CSAP/ISO 27001/ISMS 증적 자료 자동 생성
  - VS Code 플러그인 마켓플레이스 배포
  - 탐지 정확도 90% 이상
  - 자동 패치 승인율 60% 이상
  - ARR 3억원 달성 궤도 진입

### 상세 일정

| 주차 | 작업 내용 | 관련 기능 |
|------|-----------|-----------|
| Week 27-32 | GitLab Integration + Bitbucket App 개발 + 플랫폼별 테스트 | F-09 |
| Week 31-38 | CISO 리포트 엔진 개발 (PDF 생성, 인증 포맷, 자동 발송) | F-10 |
| Week 35-44 | VS Code 익스텐션 개발 (실시간 하이라이팅, 인라인 패치) | F-11 |
| Week 45-48 | 엔터프라이즈 기능 안정화 + 성능 최적화 | 전체 |
| Week 49-52 | v1.0 릴리즈 준비 + 문서화 + 마케팅 자료 | 전체 |
