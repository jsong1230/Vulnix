전체 테스트를 실행하고 결과를 리포트합니다.

## test-runner 에이전트 호출

### 전체 테스트 (인자 없음)
1. 단위 테스트 전체 실행 (`npm test`)
2. 통합 테스트 (API ↔ DB, 서비스 간)
3. E2E 테스트 (`npx playwright test`)
4. 회귀 테스트 (기존 기능 깨지지 않는지)
5. 결과 리포트 생성 → docs/tests/full-report-{YYYY-MM-DD}.md에 저장

### 범위 지정 (인자 있음)
지정된 범위에 대해 테스트 실행

단위 테스트: {통과}/{전체} ({%})
